Getting and Cleaning Data Course Project

Information about the data produced in get_data_final.txt can be found in CodeBook.md

This R code creates a data set that summarizes the average of each measured variable for each activity type and each subject from the raw data found at:

https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip

A summary of the data can be found at:

A http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

To use the code place it in the directory created from extracting the .zip listed above, source, and run the program

The code will generate a text file named "get_data_final.txt" as the final output
